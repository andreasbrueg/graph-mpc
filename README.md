# graph-mpc

## External Dependencies

In order to successfully build the source code, the following external libraries need to be installed on the system.

* [GMP](https://gmplib.org)
* [NTL](https://libntl.org)
* [Boost](https://www.boost.org)
* [OpenSSL](https://github.com/openssl/openssl)
* [EMP Tool](https://github.com/emp-toolkit/emp-tool)

In order to use the performance gains of multi-threading, [OpenMP](https://www.openmp.org) needs to be installed and available to the build system.
